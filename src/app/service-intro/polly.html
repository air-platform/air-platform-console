人工智能使用说明

01. 获取用户api key

使用Polly bot的用户名和密码登录polly后台，点击【设置】，查看客户端的【秘钥】

02. Polly bot 后台设置

如果要正常会话，需要设置Polly 【意图】，将所有可能的【表述】添加进去； 对关键的名词添加【实体】标记； 如果需要网络调用返回结果，需要设置【网络回调】

03. API 调用

使用获取到的api key作为token 调用Polly bot api查询需要的信息，api 调用实例如下：

POST /ai/v1/query HTTP/1.1
Host: 10.71.172.25:8081
Accept: application/json
Authorization: Bearer 9bd527b0cd7e11e61a2e92353f51c504
Content-Type: application/json

{
    "sessionId": "aee76d90cda111e65b6814fecdd155ca",
    "lang": "zh",
    "body": "查询天气预报",
    "contexts": [
    {
        "name": "ctx_city",
        "lifespan": 5,
        "parameters": {
            "name": "Beijing",
             "pm25":  500,
            "population": 2000
        }
    }
    ]
}

请求响应格式：

Headers:
Content-type: application/json

{
    "speech": "您好，北京的天气： xxxx",
    "displayText": "您好，北京的天气： xxxx",
    "contextOut": ["some_context", "..."],
    "source": "online-delivery-cost-query"
}

04. 网络回调

网络回调格式：

POST https://my-service.com/action

Headers:
Content-type: application/json
//user defined headers

{
    "id": "0843fb20fd7611e6c33ea1b24b6625b4",
    "sessionId": "c2211fdeeea146c88fa6572aacf9a3d0",
    "timestamp": null,
    "result": {
        "source": "agent",
        "score": 0.65675226,
        "resolvedQuery": "上海的邮编是多少？",
        "action": {
            "name": "delivery.postalcode",
            "parameters": {
                "geo-city": "上海"
            },
            "completed": true
        },
        "fulfillment": null,
        "contexts": [],
        "metadata": null
    },
    "status": 200,
    "message": "success",
    "error": null
}

网络回调服务端响应：

Headers:
Content-type: application/json

{
    "speech": "您好，上海 的邮编是 200000",
    "displayText": "您好，上海 的邮编是 200000",
    "contextOut": ["some_context", "..."],
    "source": "online-delivery-cost-query"
}
